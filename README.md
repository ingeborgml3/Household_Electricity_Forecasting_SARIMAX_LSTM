# Household_Electricity_Forecasting_SARIMAX_LSTM
# Household Electricity Forecasting with SARIMAX &amp; LSTM  Time series forecasting of hourly household electricity consumption using UCI's  household power dataset (2006-2010). Compares Seasonal Naive baseline,  SARIMAX+Fourier (dual seasonality), and LSTM multi-step models. LSTM achieves  best performance (MAE=0.445 kW, R²=0.30).
